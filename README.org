* Introduction
A very simple Wubi input method(五笔输入法) for emacs using [[http://company-mode.github.io][~company-mode~]].
* Usage
Eval ~company-wubi.el~ and use ~C-\~ to toggle the input method.
* Features
- 使用清歌输入法的词库；
- 支持标点符号自动上屏；
- 使用 ~-/=~ 键翻页；
- 使用分号与引号键选择第二第三候选词;
- 支持临时拼音，使用 ~`~ 前缀可以临时输入拼音；
- 支持拼音反查。
